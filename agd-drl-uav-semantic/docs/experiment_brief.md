# 实验简述

本文件用于记录实验背景、假设、参数和关键结果的简要说明，便于复现实验。内容应包括但不限于：

- 系统模型设定：节点布局、信道参数、动作定义等；
- 控制算法概述：AGD‑DRL、TD3、DDPG、DRL‑JPPO 参考基线；
- 语义模块描述：DeepSC 模型结构、预训练数据集、损失函数等；
- 评价指标：语义吞吐量（SUDT）、隐蔽性违规率、自干扰功率等；
- 实验配置：训练轮数、随机种子、学习率、冻结语义阶段与联合微调阶段的差异；
- 结果分析：各算法的收敛速度、语义吞吐量和隐蔽性对比，以及存在的不足和可能的改进方向。

请在实验进行过程中逐步更新本文件，并标注出待确认的 TODO 事项，以便最终撰写论文时引用。
