# 隐蔽语义通信实验代码

本仓库用于复现论文《全双工多天线 UAV 隐蔽语义通信的联合波束成形与轨迹优化：一种注意力引导的扩散强化学习方法》中的实验。

> **重要提示：**
>
> - 请首先阅读仓库根目录下的 `AGENTS.md` 和 `codex_task_main_performance.md`，其中详细说明了各阶段任务、目录结构以及必须遵守的规则。
> - 为了方便开发，本仓库采用模块化结构，环境、语义模型、强化学习算法、基线算法、训练脚本和评估脚本分离实现，并通过配置文件参数化。

本 README 仅提供仓库的整体结构说明，详细实验流程和参数请参考 `docs/experiment_brief.md` 和 `docs/reference_notes.md`。
